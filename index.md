# Aqua Ridge QBO Bridge

Aqua Ridge QBO Bridge is a private, internal tool that connects to QuickBooks Online to support Matt Libucha's own bookkeeping workflows. It is **not** a multi-user product, is not offered as a service to others, and is not available for public sign-up.

## What it does

Aqua Ridge QBO Bridge accesses QuickBooks Online accounting data — invoices, bills, customers, vendors, journal entries, reports, and related accounting records — via the Intuit QuickBooks Online API. It is operated by Matt Libucha solely against Matt Libucha's own QuickBooks Online company.

The tool runs locally on the operator's machine as a Model Context Protocol (MCP) server. It does not host data, does not expose a network service to other users, and does not have user accounts.

## Who can use it

Only Matt Libucha and authorized personnel within Matt Libucha's organization. There is no public sign-up, no end-user UI, and no shared instance.

## Authentication

Authentication is done one time via Intuit's OAuth 2.0 flow. Tokens are stored locally on the operator's machine and used solely to call the QuickBooks Online API on the operator's behalf.
